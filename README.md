# Eco-Vida-Store

Nome: Gabriel Cutrim, RM: 568059

## Descrição

EcoVida Store é um e-commerce especializado em produtos sustentáveis e ecológicos, criado para promover um estilo de vida mais consciente e sustentável. O site apresenta produtos que respeitam o meio ambiente, incentivando práticas de consumo responsáveis e ecológicas.

## Tipos de Produtos

Roupas e acessórios sustentáveis

Produtos de beleza e cuidados pessoais naturais

Itens para casa sustentáveis

Tecnologia verde

## Objetivo do Projeto

Desenvolver uma interface responsiva de e-commerce utilizando HTML, CSS, Grid Layout e Bootstrap, com foco em organização, usabilidade e experiência visual agradável, adequada para diferentes dispositivos (desktop, tablet e mobile).

## Componentes do Projeto
1. Página Inicial (Home)

Header: Barra de navegação com logo, menu e ícone de carrinho.

Banner: Carrossel usando Bootstrap, destacando promoções e produtos em destaque.

Seções de Produtos: Grid de produtos organizados em categorias (ex: Roupas, Beleza, Casa, Tecnologia), utilizando cartões do Bootstrap.

2. Página de Categorias

Grid de produtos com imagem, nome, preço e botão "Adicionar ao Carrinho".

Filtros laterais (usando colunas do Bootstrap) para refinar por preço, tipo de produto e marca.

3. Página de Detalhes do Produto

Exibição detalhada de cada produto: imagem em destaque, descrição completa, preço e botão de ação.

4. Página de Contato

Formulário estilizado com Bootstrap contendo campos: nome, email, assunto e mensagem.

Layout responsivo para uso em qualquer dispositivo.

5. Footer

Informações de rodapé, incluindo links para redes sociais.

## Tarefas de Desenvolvimento

Estruturação de Grid: Organização das seções e componentes utilizando CSS Grid e Bootstrap.

Responsividade: Ajuste do layout para diferentes tamanhos de tela, incluindo margens, espaçamentos e fontes.

Uso de Componentes do Bootstrap: Implementação de cards, carrosséis, botões e formulários para consistência visual.

Estilização Personalizada: Sobrescrita de estilos do Bootstrap para criar uma identidade visual única, mantendo cores, tipografia e efeitos de hover personalizados.

## Ferramentas e Tecnologias

Bootstrap: Framework CSS para desenvolvimento web responsivo.

Font Awesome: Biblioteca de ícones para web.

Google Fonts: Biblioteca de fontes gratuitas (ex.: Poppins).

HTML5 & CSS3: Estrutura e estilização do site.

CSS Grid e Flexbox: Organização e layout dos elementos da página.
